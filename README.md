## 📋 Sobre o Projeto

O **Synapse Mission Control AI** é um sistema de monitoramento autônomo para uma missão espacial experimental.

Utilizando a API do **Google Gemini** para processamento inteligente, o sistema analisa dados simulados de telemetria e fornece diagnósticos automáticos com recomendações de ação imediata.

O painel monitora continuamente:

* Temperatura da nave
* Nível de energia
* Status da comunicação

Além disso, o sistema identifica situações críticas, exibe alertas em tempo real e utiliza Inteligência Artificial para gerar diagnósticos objetivos sobre o estado da missão.

O projeto foi desenvolvido em **Python**, utilizando **Streamlit** para a interface visual e **Google Gemini** para a análise inteligente dos dados.

## ✨ Funcionalidades

* Simulação automática de ciclos da missão
* Inserção manual de dados de telemetria
* Monitoramento de temperatura, energia e comunicação
* Identificação automática de estados críticos
* Diagnóstico inteligente utilizando IA
* Histórico dos últimos diagnósticos realizados
* Interface visual intuitiva para acompanhamento da missão
